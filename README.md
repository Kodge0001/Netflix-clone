A high-fidelity, responsive UI clone of the Netflix India landing page. This project focuses on pixel-perfect CSS layouts, modern design patterns, and smooth interactive elements.

## 📺 Features
- **Pixel Perfect UI**: Replicates the authentic Netflix India homepage experience.
- **Responsive Layout**: Seamlessly adjusts to mobile, tablet, and desktop screens.
- **Interactive FAQ**: Custom-built accordion for the frequently asked questions.
- **Dynamic Backgrounds**: Uses high-quality assets to mirror the cinematic feel of the platform.

## 🛠️ Built With
- **Semantic HTML5**: For structured and accessible content.
- **Custom CSS3**: Utilizes Flexbox, Grid, and CSS Variables for modern styling.
- **Vanilla JavaScript**: Handles basic UI interactions and accordion logic.
